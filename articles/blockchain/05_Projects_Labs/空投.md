# 空投

## Airdrop 合约（Solidity 103 课）

### What（它是什么）

- "Airdrop 合约"用于批量将 ERC20 代币发送到多个地址，是一种链上推广与激励工具
- 支持一次调用向多账户分配不同数量的代币

### Why（为什么使用）

- 自动化批量发币，提升效率
- 比手动转账节省 gas，无需逐个调用 transfer
- 用户体验更统一，易于推广与社区运营

### How（如何实现）

合约函数 `multiTransferToken(address token, address[] addresses, uint256[] amounts)`：

1. 接收资产 token 地址、接收者地址数组、对应发币数量数组
2. 使用 `getSum(amounts)` 计算总发币量
3. `require(addresses.length == amounts.length)`
4. `require(token.allowance(msg.sender, this) >= totalAmount)`
5. 循环调用 `token.transferFrom(msg.sender, addresses[i], amounts[i])`

辅助函数 `getSum(uint256[] calldata arr)`：计算数组总和，用于校验授权余额

示例代码片段：

```solidity
function getSum(uint256[] calldata _arr) public pure returns(uint sum) {
  for(uint i = 0; i < _arr.length; i++)
    sum = sum + _arr[i];
}

function multiTransferToken(
  address _token,
  address[] calldata _addresses,
  uint256[] calldata _amounts
) external {
  require(_addresses.length == _amounts.length, "Lengths of Addresses and Amounts NOT EQUAL");
  IERC20 token = IERC20(_token);
  uint _amountSum = getSum(_amounts);
  require(token.allowance(msg.sender, address(this)) >= _amountSum, "Need Approve ERC20 token");

  for (uint256 i = 0; i < _addresses.length; i++) {
    token.transferFrom(msg.sender, _addresses[i], _amounts[i]);
  }
}
```

### 安全建议

- 校验输入数组长度一致
- 校验发币者已授权足够额度
- 循环内调用 gas 控制，不建议一次处理过多地址（可拆分批量）
