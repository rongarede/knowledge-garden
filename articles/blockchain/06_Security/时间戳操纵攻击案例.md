# 时间戳操纵真实攻击案例

## 链上博彩合约

- 利用 block.timestamp % N 产生中奖号码
- 矿工通过调整时间戳选择自己中奖

## 期权/清算时间操纵

- 清算逻辑基于 block.timestamp
- 验证者通过调整出块时间，延后或提前清算

## 预言机 TWAP 攻击

- 在低流动性池维持异常价格
- 配合时间戳调整操纵 TWAP 喂价

## 共同点

- 均利用了区块时间戳的可调范围
- 导致资金流向操纵者

## 前瞻与防御

- 避免在关键逻辑中直接依赖 block.timestamp
- 采用外部可信时间源或多源聚合
