# 荷兰拍卖（Dutch Auction）

**定义**：一种自上而下逐步降价、先到先得的拍卖机制，价格随时间递减，直到有人接受当前价格成交为止。

## 机制原理

- 设置起始价格（startPrice）
- 设置最低价格（reservePrice）
- 设置拍卖时长（auctionDuration）
- 价格随时间线性或指数下降
- 用户可在任意时间"抢购"当前价格的代币或 NFT

## 与其他拍卖模式对比

- 英式拍卖：价格从低到高，出价者相互竞价，最高者得标
- 荷兰拍卖：价格从高到低，先出价即成交，无需竞价

## 应用场景（区块链）

- NFT 初始铸造发行（如 Azuki、ArtBlocks）
- 去中心化 Token 发售（Fair Launch）
- 降低抢购 gas war 风险
- 实现链上自动价格发现

## 优点

- 成交迅速，避免多轮竞价
- 减少抢购时的 Gas War
- 公开透明的市场定价
- 合约逻辑清晰、易于实现

## 缺点与挑战

- 用户可能犹豫等待更低价，导致无成交
- 被机器人监控，捡漏不公平
- 需要合理设计初始价格和递减策略

## Solidity 实现简要示例

```solidity
function getCurrentPrice() public view returns (uint256) {
    uint256 elapsed = block.timestamp - auctionStartTime;
    if (elapsed >= auctionDuration) return reservePrice;
    uint256 priceDrop = priceDropPerSecond * elapsed;
    return startPrice - priceDrop;
}
```

## 实际使用项目

- Azuki NFT：每分钟价格下降，直至售罄
- Zora Protocol：支持多种链上拍卖方式
- Mirror Crowdfund：创作者基于荷兰拍卖模式众筹
