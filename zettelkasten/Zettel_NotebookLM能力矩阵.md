# NotebookLM 能力矩阵

> [!abstract] 核心观点
> NotebookLM 与 Obsidian 是互补工具：NotebookLM 擅长深度分析与内容生成，Obsidian 擅长知识沉淀与长期积累。

## 笔记内容

### NotebookLM 核心能力（Obsidian 难以替代）

| 能力 | 说明 | 典型场景 |
|------|------|----------|
| **YouTube 视频分析** | 自动提取字幕，支持深度问答 | 技术演讲、播客分析 |
| **多源交叉问答** | 基于多文档 RAG 检索式对话 | 文献综述、竞品调研 |
| **Podcast 音频生成** | 将资料转化为双人对话式播客 | 通勤学习、内容分发 |
| **视频/幻灯片生成** | 自动生成可视化内容 | 汇报材料、知识分享 |
| **Web 深度研究** | 自动搜索网络资料并整合 | 快速调研陌生领域 |
| **Quiz/Flashcard** | 自动生成测验和记忆卡片 | 学习巩固、知识检验 |

### Obsidian 核心能力（NotebookLM 难以替代）

| 能力 | 说明 | 典型场景 |
|------|------|----------|
| **原子化笔记** | 单一概念单一笔记，永久保存 | 知识积累、长期复利 |
| **双向链接** | `[[wikilink]]` 建立知识网络 | 概念关联、知识图谱 |
| **本地优先** | 纯 Markdown，离线可用 | 隐私敏感、长期归档 |
| **Dataview 查询** | 动态聚合笔记，构建 MOC | 知识索引、项目管理 |
| **插件生态** | 无限扩展能力 | 个性化工作流 |

### 能力定位

- **NotebookLM**：信息处理层（输入 → 分析 → 生成）
- **Obsidian**：知识沉淀层（提炼 → 链接 → 积累）
