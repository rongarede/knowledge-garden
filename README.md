# Knowledge Garden 🌱

个人知识花园，记录区块链、AI 工具链、计算机基础等领域的学习笔记与技术文章。

## 目录结构

```
zettelkasten/    原子化知识卡片（Zettelkasten 方法）
articles/
├── blockchain/  区块链：EIP/ERC 标准、DeFi 协议、Solana、安全
├── ai/          AI 工具链：Claude Code 工作流、Skill 开发
└── tech/        技术文章：工具使用、工作流分享
```

## Zettelkasten

每张卡片聚焦一个概念，使用 `[[wikilink]]` 互相关联，遵循原子化原则。
